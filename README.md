# Zodiac Name List UI

A modern and reusable user interface component for displaying names in a visually appealing list format. The design is inspired by a subtle cosmic aesthetic, combining simplicity with contemporary styling techniques.

---

## Overview

This project provides a clean and elegant way to present names using a glass-like card design. Each list item is styled consistently, ensuring that any new name added automatically follows the same visual pattern without additional configuration.

The interface focuses on readability, smooth interaction, and a balanced layout suitable for various use cases.

---

## Features

* Modern glassmorphism-based design
* Smooth hover interactions
* Fully dynamic list rendering
* Responsive and centered layout
* Easy to customize and extend

---

## Technologies Used

* HTML5
* CSS3 (Flexbox, Transitions, Backdrop Filter)

project/

---

## Usage

1. Download or clone the project files
2. Open `index.html` in a web browser
3. Add names inside the list element

Example:

```html
<ul>
  <li>Ali Maher (Zodiac)</li>
  <li>John Doe</li>
  <li>Sarah Ahmed</li>
</ul>
```

All elements will automatically inherit the defined styles.

---

## Customization

The design can be easily adjusted to fit different preferences:

* Modify background colors in the `body` selector
* Adjust glow and shadow effects using `box-shadow`
* Update typography via `font-family`
* Refine animations using `transition` and `transform`
